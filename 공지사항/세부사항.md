## 서로간의 약속

- 스터디 불참은 최소 24시간 전에 알려주기
- 서로의 질문과 답변에 대한 존중
- 메세지 확인 시 체크 표시로 알려주기

<br>

## 기록 방법

1. 해당 레포지토리를 Fork 진행
2. Sync fork로 맞춰주기
3. 스터디기록 폴더에 `2023-00-00.md` 파일로 기록
4. PR 진행

<br>

### Github

- 해당 주차 스터디 서기가 Github issue 등록
- 주제 관련 참고 자료를 댓글에 등록
- 진행한 스터디 내용을 서기가 [기록 방법](#기록-방법)으로 기록
